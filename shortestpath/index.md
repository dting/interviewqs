# Example

**Given an arr `[1, 3, 2, 1, 2, 0, 3, 1]`**

```
Possible destinations for each indices:
0: 1
1: 2, 3, 4
2: 3, 4
3: 4
4: 5, 6
5:
6: 7, done
7: done
```

```
Paths:
0   1   2   3   4   5   6   7   segments
X > X > X > X > X ----> X > X > 7
X > X > X > X > X ----> X ----> 6
X > X > X ----> X ----> X > X > 6
X > X > X ----> X ----> X ----> 5
X > X ----> X > X ----> X > X > 6
X > X ----> X > X ----> X ----> 5
X > X --------> X ----> X > X > 5
X > X --------> X ----> X ----> 4     <- shortest
```
